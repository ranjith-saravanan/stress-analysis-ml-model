# Dashboard App Prompt (Streamlit)

Build a Streamlit app with the following features:
- Multi-tab layout: Overview, Correlation, Regression, Clustering, Prediction, Personal Insights.
- CSV upload and per-user selection.
- Interactive Plotly charts with zoom & hover.
- Controls to simulate interventions (e.g., reduce Screen_Time by X minutes) and show predicted changes.
- Export functionality to download personalized recommendations as JSON.

Deliverables:
- streamlit_app.py with clear layout and modular helper functions.
- requirements.txt and a small Dockerfile for deployment.
